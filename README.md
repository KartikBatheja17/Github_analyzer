# GitHub Profile Analyzer

A full-stack web application that analyzes GitHub profiles and provides insights.

## Features
- Profile analysis
- Repository statistics
- Language breakdown
- Top repositories


## Tech Stack
- Django (Backend)
- React + Vite (Frontend)
- Axios
- GitHub REST API

## Installation

### Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
cd frontend
npm install
npm run dev
